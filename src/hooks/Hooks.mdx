# Hooks

React Pixi now support [React hooks](https://reactjs.org/docs/hooks-reference.html) ðŸŒŸ

## useApp

Access the `PIXI.Application` in your component

```jsx
import { useApp } from '@paolotozzo/react-pixi'

function MyComponent () {
  const app = useApp()
  // app => PIXI.Application

  return (
    ...
  )
}
```

## useTick

Bind listener to the `PIXI.Application` ticker event

```jsx
import { useTick } from '@paolotozzo/react-pixi'

function MyComponent () {
  useTick(delta => {
    // do something here
  })

  return (
    ...
  )
}
```
